# 📋 Incident Report Template

> Copy this template when an incident is declared. Fill in real-time during the incident.

---

## Incident Summary

| Field | Details |
|-------|---------|
| **Incident ID** | INC-YYYY-MM-DD-NNN |
| **Severity** | SEV-1 / SEV-2 / SEV-3 / SEV-4 |
| **Status** | 🔴 Active / 🟡 Mitigated / 🟢 Resolved |
| **Incident Commander** | @name |
| **Communications Lead** | @name |
| **Operations Lead** | @name |
| **Start Time (UTC)** | YYYY-MM-DD HH:MM UTC |
| **Detection Method** | Alert / Customer Report / Internal Testing |
| **Impact** | Brief description of user impact |

---

## Impact Assessment

- **Users Affected:** Number or percentage
- **Services Affected:** List affected services
- **Revenue Impact:** Estimated (if applicable)
- **Data Impact:** Any data loss or corruption?
